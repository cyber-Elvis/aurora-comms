# Topology Photo Sources

These images are downloaded web product photos that were transformed into Aurora lab node cards with overlays for node name, platform, role, and deployment domain.

The photo cards are visual stand-ins for deployment intuition only. Cisco IOL, CSR1000v, IOSv, IOS-XRv, and XRv9000 are virtual lab nodes; the physical chassis photos represent the closest real-world router class.
